JS é fracamente tipada, quer dizer que a variável não precisa receber um tipo específico de dado. Uma variável "nome" por exemplo pode receber inicialmente string, mas ser mudado para number sem mudar seu tipo

Tipos de Variáveis

  • string
  • number - int, float, infinity, -infinity, NaN 'not a number'
  • boolean - true ou false
  • undefined - o js não consegue definir seu tipo
  • null - variável vazia
  • object - arrays, matrizes, etc...

Declarando variáveis

  • var - variável escopo global, pode ser usada antes de sua declaração no código
  • let - variável escopo local
  • const - constante, não pode ser alterado, escopo local. O const pode ser alterado quando, por exemplo, a constante for um array. Pq o que é constante é a referência da variável, não seu conteúdo

Funções Nativas

  • console
    • console.log()
    • console.error()
    • console.warm() - alerta
    • console.table() - funciona melhor com arrays
    • console.clear()
  • parseFloat - transforma string em number
  • parseInt - transforma string em inteiro
  • Prompts e inputs sempre retornam string, então se for para realizar um cálculo, deve-se converter para um number
  • Math
    • Math.floor(variável) - arredonda pra baixo
    • Math.round(variável)
    • Math.ceil(variável) - arreronda pra cima

Operadores Aritiméticos

  • + : mais
  • - : menos
  • * : multiplica
  • / : divide
  • % : resto de
