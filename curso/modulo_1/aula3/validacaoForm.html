Atributos de validação

  • Atributo required: pode ser utilizado tem input e select
  • Atributo type: usado para validar entrada (email, number, date)
  • Atributos min e max: especificar valores minimos e máximos em campos numericos
  • Atributo pattern: Validar formatos de dados expecificos
  • Atributo maxlength: limintar o número máximo de caracteres em campos de texto
  • Atributo autocomplete: especifica se um campo deve ser preenchido automaticamente
  • Atributo step: definir incrementos permitidos
  • Atributo list: O elemento datalist é usado para criar uma lista de opções que aparecem como sugestões quando o usuário começa a digitar em um campo de texto. O usuário pode escolher uma opção da lista de sugestões ou continuar digitando. O datalist não cria uma lista suspensa permanente; em vez disso, ele fornece sugestões à medida que o usuário digita.
